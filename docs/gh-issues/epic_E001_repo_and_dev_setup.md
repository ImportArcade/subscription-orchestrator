# Epic E001 — Repo & Dev Setup

Goal: Create a reproducible Node.js + TypeScript developer environment and baseline repo layout so feature work can proceed.

Epic ID: E001

Associated User Stories:
- S001 — Project scaffold (docs/gh-issues/story_S001_project_scaffold.md)
- S002 — CI pipeline (GitHub Actions) (docs/gh-issues/story_S002_ci_pipeline.md)
- S003 — Docker & local dev scripts (docs/gh-issues/story_S003_docker_and_local_dev.md)
- S004 — Linting and formatting (docs/gh-issues/story_S004_linting_and_formatting.md)

Description:
Provide the repository-level setup required for development, testing, and CI. This includes TypeScript config, package.json scripts, a minimal README for contributors, local dev containers or Docker compose for dependencies, and CI automation that validates the build and tests.

Acceptance Criteria:
- All associated stories are completed.
- A developer can clone the repo and follow README steps to run tests and build the project locally.

Priority: High

Estimate: 3d (total for epic; stories have individual estimates)
