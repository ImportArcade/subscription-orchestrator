# Story S028 — In-app notification center (mobile)

Parent Epic: E009 — API & Mobile Apps
Story ID: S028

Description:
Provide an in-app notification center API and persistence that mobile clients can poll or receive via push to display reminders and action history.

Acceptance Criteria:
- API endpoints exist to list, dismiss, and acknowledge in-app notifications.
- Notifications persist in DB and are queryable by mobile clients.

Priority: Medium
Estimate: 2d

Notes:
- Consider push notifications integration as a follow-up story; start with in-app and email reminders for MVP.
